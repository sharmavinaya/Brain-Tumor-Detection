# 🧠 Brain Tumor Detection using CNN

A Deep Learning-based Brain Tumor Detection system that classifies MRI brain images into **Tumor** and **No Tumor** categories using a **4-layer Convolutional Neural Network (CNN)**. The model achieves **93% classification accuracy**, demonstrating the effectiveness of CNNs in medical image analysis.

---

## 📌 Overview

Brain tumor diagnosis from MRI scans is a challenging and time-sensitive task. This project leverages Convolutional Neural Networks (CNNs) to automatically identify the presence of brain tumors from MRI images, assisting in faster and more accurate diagnosis.

## 🏗️ CNN Architecture

The model consists of a custom **4-layer Convolutional Neural Network** with:

- 4 Convolutional Layers
- ReLU Activation
- Max Pooling Layers
- Flatten Layer
- Fully Connected (Dense) Layers
- Dropout for Regularization
- Sigmoid/Softmax Output Layer

---

## 📂 Dataset

The model is trained using MRI brain scan images categorized into:

- No Tumor

The dataset is split into:

- Training Set
- Validation Set
- Testing Set
